FAQ

HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

HTML tags are the building blocks of HTML. They define the structure and content of web pages.

CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages.

CSS can be included in HTML in three ways: inline, internal, and external.

JavaScript

JavaScript is a programming language that is used to create dynamic and interactive web content.

The DOM (Document Object Model) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

Python

Python is a high-level, interpreted programming language known for its readability and broad applicability.

Python libraries are collections of functions and methods that allow you to perform many actions without writing your code. Examples include NumPy, Pandas, and Matplotlib.

SQL

SQL stands for Structured Query Language. It is used to communicate with databases to perform tasks such as retrieving, updating, and deleting data.

A primary key is a field in a table which uniquely identifies each row/record in that table.
